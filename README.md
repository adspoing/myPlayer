# myPlayer
Player Using React+Redux


## 界面
![](http://ww1.sinaimg.cn/large/81fc3c30gw1f65g6x51hrj209109b3z1.jpg)

* 上一首，下一首
* 唱片旋转
* 开始，暂停按钮
* 播放模式，分为，随机播放，顺序播放和单曲循环
* 歌曲列表

## usage (for dev)

```text
$ npm install
$ npm start

open http://localhost:3000
```

## usage (for production)

```text
$ npm install
$ npm run build
```
all dist file lies in /dist/
